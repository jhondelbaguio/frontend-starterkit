## Structure
```
├───Assets -> Source files for the project
│   ├───images -> Images
│   ├───js -> Javascript files
│   ├───Sass -> Sass Styles 
│   │   └───main.sass -> Main file in where all other sass files should be included. Used Borboun for better SASS Experience
├───.bowerrc -> Define where the dependencies will be installed
├───.package.json -> NPM Configuration for node dependencies
├───.bower.json -> Bower Configuration file for vendor dependencies
├───.nodemon.json ->Nodemon Configuration incase be used in Node Apps
├───.jshint -> JSHint Configuration
├───.jscsrc -> JSCS Configuration for Code Style Checker
├───gulpfile.js -> Gulp Tasks
├───config.js -> Site Configuration 
```
